## 在应用内检查更新

<div align="center">
    <img src="./images/update-check.png" alt="检查更新界面" width="300"/>
</div>

1. 点击顶栏的 **设置按钮**，打开设置面板
2. 切换到 **关于** 标签页
3. 点击 **检查更新** 按钮

应用会自动比较当前版本与最新版本。如果有新版本可用，会显示更新提示。

---

## 更新源设置

默认更新源为 GitHub 仓库地址。如果你在特定网络环境下无法访问 GitHub，可以在 **关于** 标签页中修改更新源地址。

---

## 手动更新

如果自动检查更新不可用，你可以手动更新：

1. 前往 GitHub Releases 页面下载最新版本
2. **Windows**：运行新版安装包，会自动覆盖旧版本
3. **macOS**：将新版本拖入 Applications 文件夹替换旧版
4. **Linux**：用新版本的 AppImage 替换旧文件

> [!TIP]
> 更新不会影响你的设置和对话历史，这些数据存储在用户数据目录中。

---

## 数据保留

更新时以下数据会被保留：

| 数据 | 存储位置 | 是否保留 |
|------|---------|---------|
| 应用设置 | 用户数据目录 | ✅ 保留 |
| 对话历史 | SQLite 数据库 | ✅ 保留 |
| 插件配置 | 用户数据目录 | ✅ 保留 |
| 插件权限 | 用户数据目录 | ✅ 保留 |
| 自定义模型 | models 目录 | ✅ 保留 |
