## ✨ 特性概览

- 🎭 **Live2D 交互** - 高质量 Live2D 模型渲染与动画控制。
- 💬 **智能大脑** - 与后端 AI Agent 通信，实现自然语言交互。
- 🔒 **模型保护** - 业界领先的模型保护方案，支持加密下载与授权访问。
- 🌐 **跨平台** - 一套代码支持 Windows, macOS 和 Linux。

---

## 🛠️ 快速运行

```bash
# 1. 安装依赖
npm install

# 2. 编译代码
npm run compile

# 3. 启动应用
npm run dev
```
