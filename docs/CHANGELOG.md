## 1.0.2 - 2026.02.20
### ✨ 功能
- 检查更新若发现新版本后，提供新版本的更新日志预览功能，帮助用户了解新版本的改动内容
- 前端支持图片/文件回复渲染，内置后端 Agent 进行相应的协议适配
- 内置 Agent 新增 Planning 插件，提供基于 LLM 的任务规划能力，支持多步任务分解和执行，并支持 Sub-Agent 的创建和管理
- 内置 Agent 新增 Scheduler 插件，提供基于时间的任务调度能力，支持一次性和周期性任务，并支持与 Planning 插件配合使用
- 内置 Agent 新增 Image Generation 插件，提供基于 LLM 的图像生成能力，支持多种图像生成模型，并提供统一的接口供其他插件调用
- 内置 Agent Provider 模块新增获取供应商配置的接口，供插件获取特定供应商实例的详细配置信息以实现更复杂的交互逻辑
- 内置 Agent Provider 模块支持图片/文件等多模态内容的处理和传递，提供统一的接口供插件使用

### 🐛 修复
- 修复了语音自动发送时不显示发送后的输入内容的问题

### 🔧 修改
- 将内置 Agent Personality 插件的提示词中的人格设定和回复格式规范合并到一起作为人格设定暴露给用户设置，并且修改为使用长文本输入框以方便编辑
- 将所有长文本输入框修改为使用 VSCode 的 Monaco Editor 组件，提供更好的编辑体验和语法高亮支持
- 统一了内置 Agent 插件与前端插件的配置格式规范

## 1.0.1 - 2026.02.17
### ✨ 功能
- 新增 ASR 模型选择功能，支持在设置面板选择不同的语音识别模型
- ASR 模型不可用时对话 UI 麦克风图标显示灰色，并提供错误提示
- 新增应用使用帮助文档，可通过设置面板访问
- 前端插件新增文档读写工具，允许插件读写文档内容
- 内置 Agent 新增 [Agent Skills](https://claudecn.com/docs/agent-skills/) 支持，提供技能注册、调用和管理功能

### 🐛 修复
- 修复了部分资源打包路径的问题

### 🔧 修改
- 修改了内置 Agent info 插件的指令逻辑
- 修改了内置 Agent 桥接插件的工作逻辑，现在其动态解析前端插件注册工具，支持更复杂的使用场景
- 优化了工具调用审批窗口的样式和交互，当详细信息过多时支持折叠显示
- 优化了项目链接、捐赠支持、帮助文档卡片的内容和样式
